(static) downloadFile(content, filename, extension, optionsopt)

Download content as file (client side only) Supports IE10+, chrome, firefox

Parameters:
Name Type Attributes Description
content string | Array.<ArrayBuffer> | Array.<ArrayBufferView> | Array.<Blob> | Array.<DOMString>

content to save to file. If enable blob, it should be array of ArrayBuffer, ArrayBufferView, Blob, or (utf-8)DOMString
filename string

download as filename
extension string

file extension which supports png, jpg, gif, tiff, and json. For other extensions, use options.mime to customize when options.blob is false.
options object <optional>

download by BLob and its options

Properties
Name Type Attributes Default Description
blob boolean <optional>
 false

download by BLob?
mime string <optional>
 ''

MIME type of the content for the blob, e.g., 'text/plain;charset=UTF-8'.
Source:
  • utils/download.js, line 90
Example
// will download file json_file.json  containing {a:'b'} as content
let obj = {a:'b'}
downloadFile(JSON.stringify(obj),'json_file','json')

// will download file text_file.txt containing 'plain text' as content
downloadFile('plain text','text_file','txt')
