Image

new Image(idopt, classNameopt, styleopt, src, altopt, preloadopt, timeoutopt, placeholderopt, erroropt)

A React Image Component, with preloading options

Parameters:
Name Type Attributes Default Description
id string <optional>

Container element #id
className string <optional>

Classname for the container
style object <optional>

Styles for the container
src string

Image source url
alt string <optional>

Image alt
preload boolean <optional>
 true

Allow preloading image? If false then will act as normal tag
timeout number <optional>
 30000

When preload is enabled, maximum time (in milliseconds) to wait before error kicks in
placeholder string <optional>

When preload is enabled, alternative image url to show when image load has failed
error renderable <optional>
 'Load failed'

When preload is enabled, error message to show when image load has filed
