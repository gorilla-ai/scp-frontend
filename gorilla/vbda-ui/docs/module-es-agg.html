(inner) esFormat(request, response, idSeparatoropt) → {Object}

Parses both request and response body of Elasticsearch query into Chart API Series' middle format.

Parameters:
Name Type Attributes Default Description
request Object

The object containing query (and aggs) part of Elasticsearch request body. Only the aggs part will be used.
response Object

One element from responses part of Elasticsearch response body. Currently only supports one response body. If an array is passed, only the first element will be used.
idSeparator string <optional>
 _

Separates aggregation ID with the aggregation field name.
